# Screenshots

Put your project screenshots in this folder and reference them from the main `README.md`.

Suggested filenames:

- `menu.png`
- `crud.png`
- `db-records.png`

How to take screenshots:

1. Run the app and capture the menu + CRUD operations.
2. In MySQL Workbench/CLI, run:
   - `SELECT * FROM students;`
   - screenshot the result.
3. Save images into this folder.

Then, enable the image links in `README.md` under the **Screenshots** section.
